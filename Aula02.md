## Engenharia de Prompt - Fundamentos

### Prompt Genérico

**Pergunta:**  
O que é Machine Learning?

**Resposta:**  
Machine Learning é um ramo da inteligência artificial que permite que sistemas aprendam a partir de dados e melhorem suas decisões sem serem explicitamente programadas.

---

### Prompt Estruturado

**Prompt:**  
Atue como um professor. Explique o que é Machine Learning para um iniciante usando uma analogia com esportes em dois parágrafos.

**Resposta:**  
Machine Learning é como treinar um atleta. No começo, o atleta pratica várias vezes, observa seus erros e melhora seu desempenho com base na experiência. Da mesma forma, um algoritmo de Machine Learning analisa muitos dados para aprender padrões e melhorar suas decisões.

Assim como um jogador de futebol melhora com treinos e partidas, o algoritmo também evolui conforme recebe mais dados e feedback. Com o tempo, ele se torna mais preciso ao tomar decisões ou fazer previsões.

---

### Análise Crítica

A resposta do prompt estruturado foi mais útil porque trouxe contexto, objetivo e formato claros. Isso reduziu a ambiguidade da pergunta e permitiu uma explicação mais didática e organizada, facilitando o entendimento para iniciantes.
