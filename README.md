# Engenharia_Prompt_e_Fundamentos_AI
Disciplina do Curso de Ciência da Computação (2026_1)

Este repositório é para a disciplina Engenharia de Prompt e Aplicações em Inteligência Artificial. Explorar técnicas e metodologias para projetar, otimizar e avaliar interações com modelos de linguagem e sistemas de IA generativa. O foco está em transformar ideias em instruções práticas que resultem em respostas úteis, criativas e confiáveis.
